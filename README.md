# 简单股票分析系统


## 功能
> 股票的登记
>
> 使用第三方接口，获取股票每天的交易信息并存储到数据库中进行分析
>
>  分析自选股票中处于低值的股票，并用不同颜色进行标识
>  
>  分析选定的股票，获取各种买点的价格用于购买决策
>  
## 如何获取
> 需要演示或者商务合作可以加微信（xpyzwm）

## 使用范围
> 作为管理系统基础模板使用

> 适用于毕业设计及相关产品演示
